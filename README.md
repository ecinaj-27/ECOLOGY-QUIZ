# Web Quiz Game

## Run

1. Install dependencies:
   - `npm install`
2. Start server:
   - `npm start`
3. Open:
   - Student quiz: `http://localhost:3000/`
   - Teacher dashboard: `http://localhost:3000/admin`

## Teacher Login

- Username: `teacher`
- Password: `admin123`

## Media Placeholders

Add these files:

- `public/assets/videos/q1.mp4`
- `public/assets/videos/q2.mp4`
- `public/assets/videos/q3.mp4`
- `public/assets/videos/q4.mp4`
- `public/assets/videos/q5.mp4`
- `public/assets/audio/background.mp3`

The app works without them, but add your own media to enable backgrounds and music.
